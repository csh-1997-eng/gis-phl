# investigations

Use this directory for work that sits between open exploration and formal experiments.

Place work here when:

- the question is concrete enough to organize around
- the analysis needs more structure than an exploratory notebook
- the evaluation plan is still moving
- the goal is to shape future experiments rather than make a final claim

Do not use this directory for loose scratch work. Do not use it for fully reproducible experiment runs. It is the middle stage between those two modes.
